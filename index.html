Research
Project · ROS2 · Multi-Robot Navigation
ROS2 Multi-Robot Swarm Navigation with Nav2
Arizona State University · Jan 2026
A 4-robot swarm navigation system where TurtleBot3 Waffle robots navigate cooperatively in a shared Gazebo environment using ROS2 Jazzy and Nav2. The delta movement strategy ensures all robots move by the same displacement vector as the leader, preserving formation geometry for any goal direction. Features include sequential dispatch, auto-recovery spiral search, and laser scan filtering to prevent robots from treating teammates as obstacles. Achieved 85–90% goal success rate across all tested scenarios.
